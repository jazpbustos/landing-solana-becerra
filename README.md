# Landing Page — Solana Becerra | Abogada

Landing page profesional desarrollada para Solana Becerra, abogada especializada en distintas áreas del derecho.

 ✨ Sobre el proyecto

Este proyecto fue diseñado y desarrollado como una práctica de maquetación frontend moderna, enfocada en crear una experiencia visual elegante, profesional y responsive para un estudio jurídico.

La página cuenta con:
- Diseño responsive
- Hero section profesional
- Sección de servicios con cards
- Información de contacto
- Integración visual de redes sociales
- Botón flotante de WhatsApp
- Estética minimalista y moderna

🛠️ Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
